WhatsApp Bot API

This documentation explains how to use the external API endpoints to trigger automated WhatsApp messages from your website, CMS, or other external applications.

Send Order Confirmation

POST /api/confirmation

Use this endpoint to send a WhatsApp order confirmation to a customer automatically after they place an order on your website.

Required Headers

Content-Type: application/json

Request Body (JSON)

The request body must be a JSON object containing the following parameters:

  • phone (Required): The customer's absolute phone number, including the country code without plus or symbols (e.g., "212704969534").
  • customerName (Optional): The name of the customer to personalize the message.
  • orderNumber (Optional): The unique identifier/ID for the order.

Example Request (JavaScript / Client-side)

fetch('https://996b-2a01-4f9-c013-e885-00-1.ngrok-free.app/api/confirmation', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        phone: "212704969534",
        customerName: "Mohamed Bella",
        orderNumber: "ORD-98765"
    })
})
.then(res => res.json())
.then(data => console.log('Response:', data));

Example Successful Response

{
    "success": true,
    "message": "WhatsApp message sent to client"
}

Example Error Response (Missing Phone)

{
    "success": false,
    "error": "Phone number is required"
}
